# Community News – UI Sample (HTML)

これは「特定コミュニティにのみ発信するオウンメディア・ニュースアプリ」の
**ワイヤー（最初の1枚）**を、HTML/CSS/JSでUI化したサンプルです。

## できること（UIのみ）
- Home：検索 / チャンネル（chips） / カード型フィード
- 記事詳細：要点（3つ）→本文→CTA（リンク）
- 保存：ブックマーク（localStorage）
- 通知：重要のみトグル

## 使い方
1. `index.html` をブラウザで開くだけで動きます（サーバ不要）
2. iPhone/Androidで確認したい場合は、同フォルダを端末へ送って開くか、
   ローカルサーバ（例：VSCode Live Server）で開いてください。

## 次の拡張（クライアント向け提案に便利）
- ログイン（招待制）/ 役割（運営投稿 / 承認制投稿）
- 記事データをCMSやスプレッドシート/Firestore等に接続
- Push通知（PWAならWeb Push、ネイティブならFCMなど）
